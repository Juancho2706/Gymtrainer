# Olympus Wolf - Sitio Web Personal Trainer

Plataforma web moderna y de alto rendimiento diseñada para servicios de entrenamiento personal. Centrada en una estética agresiva ("Kinetic Future"), velocidad de carga y conversión de usuarios.

## Tecnologías

-   **Core**: Next.js 16 (App Router)
-   **Lenguaje**: TypeScript
-   **Estilos**: Tailwind CSS
-   **Animaciones**: Framer Motion
-   **Linting**: ESLint



## Estructura de Carpetas

-   `/src/app`: Rutas y páginas (App Router).
-   `/src/app/components`: Componentes reutilizables (Hero, About, etc.).
-   `/public`: Recursos estáticos (imágenes, videos optimizados).
